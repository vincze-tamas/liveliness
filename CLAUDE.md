Before attempting ANY fix or configuration change, you MUST:
1) Search for and read all relevant documentation files in this repo (README, docs/, CONTRIBUTING, config examples), 
2) If the issue involves an external tool or API, use WebFetch to pull its official docs, 
3) Create a TodoWrite plan listing what you found in docs and your proposed approach, 
4) Only then begin implementation. Never guess at CLI flags, API endpoints, or config syntax—always verify from source first. 
If you catch yourself guessing, stop and say 'I need to check the docs for this.'
